# Open-Meteo Weather Tool

Free weather API for Hive agents. No API key required.

## Tools

### weather_get_current
Get current weather for any location.
- latitude (float): e.g. 52.52 for Berlin
- longitude (float): e.g. 13.41 for Berlin

### weather_get_forecast
Get multi-day forecast for any location.
- latitude (float): Location latitude
- longitude (float): Location longitude
- days (int): 1-16 days (default 7)

## Authentication
None required. Completely free.
